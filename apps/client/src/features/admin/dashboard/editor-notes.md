# Ważne informacje dla edytorów

## Zgłoszenia

Przy modyfikacji zgłoszenia, utworzonego przez inną osobę, po zmienie danych, trzeba kliknąć przycisk `Zapisz zmiany` przed zaakceptowaniem zgłoszenia, inaczej Twoje zmiany nie zostaną zapisane, a zgłoszenie będzie wrzucone do bazy "as-is"

Zmienione i nowe komórki domyślnie nie są potwierdzone. Trzeba je potwierdzić klikając na każdy checkbox, lub na przycisk `Potwierdź komórki`, i kliknąć `Zapisz zmiany` przed zaakceptowaniem zgłoszenia

## Stacje

Jest całkowity zakaz dodawania `NetWorks` w notatkach komórek. Tego samo się tyczy `[E-GSM]` - do tego jest osobny checkbox
