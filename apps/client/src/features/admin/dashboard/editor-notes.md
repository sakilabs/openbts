# Ważne informacje dla edytorów

## Zgłoszenia

Przy modyfikacji zgłoszenia, utworzonego przez inną osobę, po zmianie danych (np. Potwierdzenie komórek), trzeba kliknąć przycisk `Zapisz zmiany` przed zaakceptowaniem zgłoszenia, inaczej Twoje zmiany nie zostaną zapisane, a zgłoszenie będzie wrzucone do bazy "as-is". Notka sprawdzającego się do tego nie wlicza.

Zmienione i nowe komórki domyślnie nie są potwierdzone. Trzeba je potwierdzić klikając na każdy checkbox, lub na przycisk `Potwierdź komórki`, i kliknąć `Zapisz zmiany` przed zaakceptowaniem zgłoszenia

Proszę **uważnie** patrzeć na zgłoszenia z zdjęciami. Proszę nie akceptować zgłoszeń z zdjęciami które zostały dodane przed chwilą (na tej samej lokalizacji), tylko pokierować użytkownika w notatkach, że po zaakceptowaniu może wybrać zdjęcia dostępne na lokalizacji do danej stacji

Przy zgłoszeniach które dodają PCI do NR NSA, proszę uważnie sprawdzać czy PCI jest poprawne
